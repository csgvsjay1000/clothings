## 服务部署

### 服务列表 ###

***1、基础服务***

- zookeeper ( kafka、dubbo、分布式锁 )
- redis ( 缓存服务 )
- kafka ( 消息服务器 )

***2、应用服务***

- tag ( 标签服务：绑定、解绑、EPC查询、库存查询 )
- factory ( 工厂服务 : 工厂发货/收货)
- warehouse ( 仓库服务 : 仓库发货/收货/横调/退仓)
- warehouse-inventory ( 仓库盘点服务 )
- shop ( 门店服务 : 门店发货/收货/横调/退仓)
- shop-inventory ( 门店盘点服务 )
